# Movie

Сервис для фильмов. 
Возможности:
- Поиск фильма по базе
- Просмотр оценок других пользователей
- Коментрирование фильмов
- Оценка фильма
- Добавление в базу своих фильмов

## Использовалось:
- Java8 SE
- Spring(движок)
- Hibernate(для работы с БД)
- MySQL(сама БД)
- Thymeleaf(шаблонизатор для фронтэнда)
